== Description ==

- A plugin to migrate users ( username and password only ) from Joomla to WordPress.<br />
- This plugin migrate limited users(100) and do not migrates admin authors.
- If you want to migrate users infos like -> <strong>first name, last name, roles, display name, description, email, registration date, status and admin users with roles. </strong>
- Also migrate more database fields as requirements.

== Installation ==

1. Upload the `joomla-users-to-wp-users` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to the Migrate Joomla Users.
1. Set Host, Username and Password of joomla database same as wordpress database(Import joomla database on same as wordpress database host).
1. Delete users of wordPress site excluding admin user.
1. To remove duplication of users ID in users table, please make unique ID of admin user ID in user table and user meta table which are not exists in joomla users IDs.
